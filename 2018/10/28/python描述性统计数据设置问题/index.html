Describe输出格式设置

2018-10-28

做数据初探时,描述性统计是必不可少的方法之一。python中describe函数经常会输出讨厌的科学计数法，让人脑海里再乘个10的多少倍,早就迷糊了,特别是在数值本身位数较少的情况下。 

import pandas as pd  
import numpy as np  

test_data=pd.DataFrame(np.random.randint(1000000,10000000,size=(60,5)))  

test_data.describe()

加上如下语句即可设置describe结果的格式(也可自己设置小数据个数): 

pd.set_option('display.float_format', lambda x: '%.0f' % x)
